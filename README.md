# Frontend Drive Wise

O **Drive Wise** é uma aplicação web com interface moderna inspirada em apps como Google Maps e Waze, focada no planejamento de rotas e análise de viagens.

## Funcionalidades
-  Visualização de rotas em mapa interativo
-  Inserção de origem e destino
-  Seleção de veículo
-  Base para cálculo de consumo e custo de viagem
-  Interface responsiva estilo aplicativo mobile

## Interface

O layout foi desenvolvido com foco em experiência do usuário, incluindo:
- Mapa em tela cheia
- Painel inferior flutuante
- Header com design moderno (corte diagonal)
- Inputs centralizados e intuitivos

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Integração com Google Maps (via iframe)

## Possíveis melhorias
- Integração com API oficial do Google Maps
- Cálculo automático de distância e tempo
- Estimativa de consumo de combustível
- Modo escuro
- Geolocalização em tempo real
